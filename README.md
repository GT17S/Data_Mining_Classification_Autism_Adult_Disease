# Data_Mining_Classification_Autism_Adult_Disease
Classification des données issu d’une base de données médicale liée au Autisme pour classifier et prédire les malades et non malades.
- Manipulation de l'outil WEKA 
- Compréhension des données (Data Understanding)
- Pré-traitement des données (Data PreProcessing) 
- Visualisation des données (Data Visualization)
- Application des algorithmes de classification : Decision tree et Random Forest
